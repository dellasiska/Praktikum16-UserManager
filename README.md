# Praktikum 16 - User Manager

Aplikasi User Management berbasis Vue 3 yang menggunakan MockAPI sebagai backend.

## Teknologi yang Digunakan

- Vue 3
- Vite
- Vue Router
- Axios
- Tailwind CSS
- MockAPI

## Fitur

- Menampilkan daftar user
- Menampilkan detail user
- Menambahkan user
- Mengubah data user
- Menghapus user

## API

Project ini menggunakan MockAPI sebagai backend untuk proses CRUD (Create, Read, Update, Delete).
